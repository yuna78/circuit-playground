# 电路乐园（Circuit Playground）MVP 提案

## Why

孩子正在学初中物理电学（串并联、电压、电流、欧姆定律等），抽象的"看不见的电"靠课本和公式难以建立直觉。现有工具中：PhET 电路仿真是纯沙盒、无游戏化目标与进阶引导；Falstad circuitjs 面向工程师、对孩子不友好。需要一个**卡通画风 + 闯关引导 + 电流电压可视化**三者兼备的中文互动游戏，让孩子通过拖拽元件搭电路、实时观察电流电压变化来理解电学知识。

## What Changes

- 从零创建一个纯前端 SPA 网页游戏（无后端），部署到 Cloudflare Pages
- 电路搭建画布：从元件抽屉拖拽元件到画布、导线连接、网格吸附、删除/旋转
- 元件库（MVP 九种）：电池、灯泡、定值电阻、滑动变阻器、开关、电压表、电流表、保险丝、导线
- 通用电路解算器：改进节点分析法（MNA），任意接法（含桥式、多回路、短路、断路）均可求解
- 可视化三件套：电流 = 流动小点（密度/速度表大小与方向）；电压 = 电势颜色梯度；元件状态 = 灯泡亮度随功率渐变、电阻发热、保险丝过流烧断
- 双模式：自由实验室（全元件解锁的沙盒）+ 闯关挑战（8 章关卡线，初中打底、末两章高中入门：电源内阻与动态分析）
- 关卡系统：关卡为 JSON 数据（可用元件、目标、判定条件、星级），进度存 localStorage
- 全中文界面，术语与国内初中物理课本一致
- 卡通玩具风 SVG 美术素材（美术方向已定调：黏土玩具质感、圆润、部分元件带表情）

## Capabilities

### New Capabilities

- `circuit-solver`: 电路解算内核——MNA 节点电压法纯函数模块，输入电路拓扑与元件参数，输出各节点电势、各支路电流、各元件功率；处理短路/断路/开关态/保险丝熔断/电表内阻等边界
- `circuit-canvas`: 电路搭建画布——元件拖拽、网格吸附、端点连线、选中/删除/参数编辑等交互
- `circuit-visualization`: 电学量可视化——电流小点流动动画、电势颜色梯度、灯泡亮度/电阻发热/保险丝烧断等元件状态渲染、电表读数显示
- `component-library`: 元件库——九种元件的定义（参数、端点、行为模型）与卡通 SVG 视觉（含各状态视觉）
- `sandbox-mode`: 自由实验室——全元件解锁的沙盒模式，随意搭建与观察
- `level-mode`: 闯关挑战——8 章关卡线、关卡目标判定、星级评价、进度保存与元件逐步解锁
- `app-shell`: 应用外壳——模式选择首页、全中文 UI、路由、部署配置（Cloudflare Pages）

### Modified Capabilities

（无——全新项目，无既有 spec）

## Impact

- 全新代码库：Vite + React + TypeScript 前端项目（本仓库当前为空）
- 无后端、无账号体系；进度数据仅存浏览器 localStorage
- 部署依赖：Cloudflare Pages（静态构建产物）
- 美术素材：手写 SVG 组件（参考已生成的黏土玩具风格定调图）；无运行时图片资源依赖
- 面向桌面/平板浏览器（鼠标与触摸拖拽），不做手机小屏适配
