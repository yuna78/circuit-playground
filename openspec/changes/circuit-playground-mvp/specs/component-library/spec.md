# component-library 元件库

## ADDED Requirements

### Requirement: MVP 九种元件
系统 SHALL 提供以下元件，各含电学模型参数、端点定义与卡通 SVG 视觉：电池（电动势、内阻默认 0）、灯泡（额定电压、额定功率）、定值电阻（阻值）、滑动变阻器（最大阻值、滑片位置）、开关（通/断）、电压表（10MΩ 内阻）、电流表（理想 0 内阻）、保险丝（额定电流）、导线。

#### Scenario: 元件注册完备
- **WHEN** 打开元件抽屉
- **THEN** 九种元件全部可见、可拖拽，中文名称与课本一致

#### Scenario: 元件默认参数合理
- **WHEN** 拖入一个新灯泡
- **THEN** 默认参数为课本口径（如"2.5V 0.3W"级别），接 1–2 节干电池即可正常点亮

### Requirement: SVG 视觉与状态
每个元件 SHALL 为独立 SVG React 组件，风格遵循黏土玩具定调（圆润、柔和渐变、部分元件带表情），并通过 props 呈现状态（灯泡亮度级、开关通断、保险丝完好/熔断、电池发烫、变阻器滑片位置、电表指针角度）。

#### Scenario: 状态驱动外观
- **WHEN** 传入 brightness=0.8 给灯泡组件
- **THEN** 呈现 80% 亮度辉光效果

#### Scenario: 矢量缩放
- **WHEN** 画布缩放 50%–200%
- **THEN** 元件边缘清晰无锯齿无模糊

### Requirement: 端点元数据
每种元件 SHALL 声明其端点数量、相对位置与极性（电池 +/−、电表 +/− 接线柱），供画布连线与解算网表编译使用。

#### Scenario: 极性端点
- **WHEN** 查询电池元件定义
- **THEN** 返回正极与负极两个端点及其网格相对坐标
